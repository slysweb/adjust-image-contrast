Privacy Policy

Adjust Image Contrast
Last updated: July 17, 2026

This Privacy Policy describes how the browser extension Adjust Image Contrast (“the Extension”) handles information. The Extension’s only purpose is to let users select an image on a webpage and adjust its contrast locally in the browser.

1. Information We Collect

The Extension does not collect personal information, browsing history, account data, or analytics. It does not send any data to remote servers. Images are never uploaded or stored on our servers.

When you use the Extension, it may temporarily process and store the following information locally on your device:

  • Whether image-selection mode is currently active
  • The contrast value you set for the selected image (session only)

2. How Information Is Used

Local session data is used only to:

  • Keep selection mode active while you adjust contrast
  • Apply a CSS contrast filter to the image you selected
  • Sync the on-page HUD and extension popup controls

This data is never sold, shared with third parties, or used for advertising, profiling, or cross-product tracking.

3. Storage and Retention

Mode state and contrast preferences are stored using the browser’s session storage APIs on your device. Data is retained only for your convenience during the browsing session and can be cleared by stopping the tool, closing the browser session, or removing the Extension.

4. Permissions

The Extension requests only the permissions needed for its core purpose:

  • activeTab — interact with the current tab when you start selecting
  • scripting — inject the selection UI into the active page
  • storage — keep selection mode state locally for the session

5. Third-Party Services

The Extension does not include analytics SDKs, advertising networks, authentication providers, or remote image-processing APIs. All contrast adjustments run locally via CSS filters in your browser.

6. Children’s Privacy

The Extension is a utility tool and is not directed to children. It does not knowingly collect personal information from anyone, including children under 13.

7. Changes to This Policy

If privacy practices change, this page will be updated and the “Last updated” date will change. Material changes will be disclosed as required by store policies.
